HTML


HTML (HyperText Markup Language - Lenguaje de marcado de hipertexto) determina la estructura del contenido de una página web. Consta de un conjunto de elementos (etiquetas) que se utilizan para, incluir o eliminar contenido, ya sea texto negrilla o cursiva, enlazar una dirección web en un texto, insertar imagenes o videos.


Hoy en dia y debido al constante cambio en el desarrollo web, actualmente se utiliza HTML5 (quinta versión de la World Wide Web), que es mucho más dinámico y contiene elementos multimedia. A diferencia del HTML nativo, que sólo permitía la creación de sitios web principalmente estáticos y sin tanto contenido dinamico.


En este articulo, daremos información sobre la estructura principal de HTML con sus etiquetas mas utilizadas, dentro de cada contenedor e información sobre estructura, anatomia, etiquetas, caracteres, comentarios, imagenes, videos, hipervínculos, etc.


  1. Estructura principal

  • !DOCTYPE

    • Informa al navegador qué versión de HTML o XML (Lenguaje de Marcado Extensible), se utiliza para escribir el documento. El DOCTYPE es una declaración y no una etiqueta. Esta es la cadena más corta que se considera un tipo de documento válido.



  • HTML

    • Encierra todo el contenido de la página HTML, se llama elemento raíz principal.



  • LANG

    • Dentro del elemento o etiqueta "HTML" se puede establecer el idioma correspondiente de la pagina web.



  • HEAD

    • Actúa como un contenedor para cualquier cosa que desee agregar a una página HTML que no sea visible para los visitantes de la página. Esto incluye palabras clave, descripción de la página en los resultados de búsqueda, CSS para aplicar estilo al contenido, declaraciones de caracteres, JAVASCRIPT y más.

  • META

    • Describe un documento generado, quién lo escribio, sirve para aportar información sobre el documento. Contiene atributos como: "NAME" (Especifica el tipo de metadato del que se trata; es decir, qué tipo de información contiene) y "CONTENT" (Especifica el contenido del metadato en sí).

  • TITLE

    • Aparece en la pestaña del navegador o en la barra de título cuando se carga la página. Se utiliza para describir la página cuando está marcada como favorita o marcada como favorita en la barra de favoritos de una navegador web (Google Chrome, Mozilla Firefox, Microsoft Edge, etc).

  • BODY

    • Registra todo el contenido visible que uno desea mostrar a los usuarios web que visiten la página web o que el usuario final solicite que el contenido que necesite sea acorde a sus necesidades, ya sea texto, imágenes, videos, juegos, sonidos y muchos más.

  • LINK

    • Esta etiqueta requiere dos atributos: rel="stylesheet" para indicar que se trata de una hoja de estilo de un documento y "href" para contener la ruta al archivo de hoja de estilo de CSS.

  • SCRIPT

    • Este es un atributo que nombra el JavaScript (lenguaje de programación interpretado) que debe cargarse y el atributo: defer que básicamente le dice al navegador que cargue el archivo de JavaScript, al mismo tiempo que el HTML de la página. Esto es útil porque hace que todo el HTML se cargue antes de que se ejecute JavaScript.

  1. Estructura de una etiqueta
  • Etiqueta de apertura, consta del nombre del elemento (en este caso p) rodeado por corchetes angulares de apertura y cierre (<>).
  • Etiqueta de cierre, es la misma que la etiqueta de apertura, excepto que incluye una barra diagonal de cierre (/) antes del nombre de la etiqueta.
  • Contenido, que uno desea agregar, debe estar dentro de las etiquetas de apertura y cierre.
  • Elemento, es la combinación de etiqueta de apertura, cierre y el contenido.
  1. Contenedores
  • HEADER

    • El encabezado por lo general, consta de una barra grande en la parte superior de la página con un título grande, un logotipo y algún texto introductorio, esta parte suele permanecer igual cuando se navega entre páginas del mismo sitio web.

  • NAV

    • La barra de navegación, son enlaces que se encuentran en la parte principal del sitio, consta normalmente de un menú con botones, enlaces o pestañas, al igual que el título, este contenido es el mismo en las diferentes páginas del sitio web.

  • MAIN

    • El contenido principal, es la parte central de la página y contiene la mayor parte del contenido específico de la página web, tiene varias subsecciones representadas por elementos article, section, div. Solo se puede utilizar una vez en la página web y se debe colocar dentro de la etiqueta body.

  • ARTICLE

    • El articulo o autocontenida, es un bloque de contenido que es útil independientemente del resto de la página, como una publicación de blog.

  • SECTION

    • La sección generica, se utiliza para agrupar cada parte de una página que forma su propia sección, como un mapa, conjunto de títulos o resúmenes.

  • ASIDE

    • La barra lateral, contiene información adicional como enlaces, anuncios, etc. Esta relacionado con el contenido principal de la página, pero en otros casos encontramos elementos repetidos, como un menú de navegación secundario.

  • FOOTER

    • El pie de pagina, se encuentra en la parte inferior de la página que generalmente contiene texto pequeño, derechos de autor, redes sociales, información de contacto, información general como asi mismo en el encabezado, pero generalmente es secundaria y no es tan importante con respecto a la página.

  • SPAN

    • Un contenedor en línea, es un elemento no semántico utilizado dentro de una línea, se utiliza cuando no podemos utilizar ningún otro elemento textual semántico para incluir el contenido.

  • DIV

    • La división de contenido, es un contenedor general que no afecta el contenido o el diseño hasta que se le aplique algún estilo con el formato de hojas de estilos CSS.

  • BR

    • Saltos de linea, se genera dentro un párrafo de la historia; es la única manera de presentar líneas cortas como una dirección postal o un poema.

  1. Multimedia

    2. HTML utiliza etiquetas para la insercción de imagenes y videos dentro de nuestra pagina WEB utlizando algunos atributos:

    • SRC (source): Contiene la ruta del archivo de imagen, la ruta o la ubicación de la imagen.
    • ALT (alternative): Contiene un texto que debería describir la imagen, y que podría ser accedido por usuarios que no pueden ver la imagen.
    • Width: Especifica el ancho de la imagen con la unidad en píxeles, se recomienda manejarlo en CSS.
    • Height: Especifica la altura de la imagen con la unidad en píxeles, se recomienda manejarlo en CSS.

    Explicaremos las 5 etiquetas para insertar elementos multimedia:

    • Imagenes: La etiqueta "IMG", se utiliza para insertar imagenes en una pagina HTML. Es una etiqueta vacia, la cual no contiene texto, sino atributos "SRC" y "ALT", no tiene etiqueta de cierre.
    • Videos: La etiqueta "VIDEO", se utiliza para mostrar videos directamente desde la pagina de HTML. Es una etiqueta vacia, la cual no contiene texto, sino atributos "SRC", tiene etiqueta de cierre.
    • Audios: La función de la etiqueta "AUDIO" funciona muy similar a la etiqueta de "VIDEO", la diferencia es que solo se utilizara archivos de audio. Es una etiqueta vacia, la cual no contiene texto, sino atributos "SRC", tiene etiqueta de cierre.
    • Tracks: La función de la etiqueta "TRACKS", se utiliza para pistas de texto cronometrados, se pueden incluir en subtitulos, titulos, descriptiones, capitulos y metadatos mediante las etiquetas de "VIDEO" e "IMG". Es una etiqueta vacia, la cual no contiene texto, sino atributos "SRC", "KIND", "SRCLANG" y "LABEL", no tiene etiqueta de cierre.
    • Picture: Muestra las imagenes mucho mas flexibles al contrario de la etiqueta "IMG" que muestra imagenes de manera tradicional y dependiendo de nuestra necesidades, se puede agrupar una serie de imagenes. Es una etiqueta vacia, la cual no contiene texto, sino atributos "SOURCE", "IMG" y "ALT", no tiene etiqueta de cierre.
  1. Marcado de Texto y Formateo de texto avanzado

    2. El marcado de texto, permiten especificar ciertas partes del encabezado, que permite especificar ciertas partes del contenido (encabezados y sub encabezados). En HTML se tiene 6 niveles de encabezado a partir de la etiqueta "H1" al "H6".

    La etiqueta "p" se utilizara frecuentemente para el marcado de contenido de texto regular.

    El marcado de texto avanzado, refiere a marcar una serie de conjuntos de elementos, como los terminos, deficiones, preguntas y respuestas

    Las citas en linea funcionan de la misma manera, excepto que se utilizan la etiqueta "Q".

    En HTML registra una etiqueta para marcar la información de un contacto, mediante la etiqueta "ADDRESS".

    Los superindices "SUP" y subindices "SUB" se utilizan para marcar elementos como fecha, formulas quimicas y ecuaciones matematicas.
