CSS


CSS que significa hojas de estilo en cascada, permite crear paginas web atractivas, en las cuales se puede utilizar junto con HTML (etiquetas), manejar un contenido, presentación de un documento principal estructurado y dinamico en los sitios web.


Estas se llaman tambien hojas de estilo secuenciales, porque se puede tener varias hojas y una de ellas hereda las propiedades de las en si demás. Si se desea cambiar el aspecto de un sitio web, se necesita implementar CSS, que junto con un buen sistema de gestión de contenidos, aumentara la cobertura de visualización de la pagina web.


En este articulo explicaremos las sintaxis y propiedades mas utilizadas en CSS.


Sintaxis
Es un conjunto de reglas que definen cómo escribir código en CSS. Estas reglas definen cómo se deben estructurar las reglas de estilo y cómo se deben combinar los selectores, propiedades y valores para producir un diseño coherente en una página web.
Selector Propiedad Valor Regla Comentario
Esta es la primera parte de una regla CSS que le indica al navegador a qué elementos HTML (etiquetas) se aplica la regla. Los selectores CSS son la base de las reglas estructurales que determinan la estética y el diseño de los elementos del sitio web.

Ejemplo: (tipo, clase, ID, atributo y universal). 		Este es un elemento de estilo especial que se puede cambiar en elementos seleccionados. Cada propiedad CSS tiene un conjunto de valores que se pueden establecer para determinar su apariencia, dependiendo del selector que elijamos.

Ejemplo: (color, border-color, margin, padding). 		Es una forma de definir el conjunto de subvalores admitidos por una propiedad. La validez del atributo depende del tipo de valor establecido y del parámetro específico.

Ejemplo: (píxeles, porcentajes, nombre del color). 		Es un conjunto de declaraciones que se aplican a uno o más elementos. Una regla CSS consta de uno o más selectores seguidos de una declaración que consta de uno o más pares de propiedad y valor.

Ejemplo: (selector + propiedad + valor). 		Se añade a un archivo CSS para informar al diseñador de su utilidad y proporcionar información. Los comentarios se pueden colocar en cualquier espacio en blanco permitido en la hoja de estilo y se insertan entre las etiquetas /* (apertura) y */ (cierre). Los navegadores web ignoran todo el texto entre estas dos etiquetas.

Ejemplo: (/* */).

Selectores
Primera regla en CSS, es un patrón de elementos y otras expresiones que le indica al navegador qué elementos HTML seleccionar para activar reglas que contienen valores de propiedad CSS.
