CSS


La abreviatura CSS significa hojas de estilo en cascada el cual se puede utilizar junto con HTML, que maneja el contenido principal de los sitios web. Estas se llaman hojas de estilo secuenciales, porque se puede tener varias y una de ellas hereda (o "secuencial") propiedades de las demás.

Para muchos, una simple plantilla de blog es suficiente. Sin embargo, si se desea cambiar el aspecto de tu sitio web se necesita implementar CSS, que junto con un buen sistema de gestión de contenidos, aumentara la cobertura del contenido.
